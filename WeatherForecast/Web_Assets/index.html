Name: WeatherForecast
Credits: Dilip Kosuri

This app contains the following features:

  • Favourites
  • Adding bookmarks by pinning on the map
  • Search the city/location using search context bar
  • View detailed info of the bookmarked location for upto 5 days of forecast
  • Settings - to change the units and temperature metrics
  • Reset app storage -- clear the locations bookmarked
  • Help screen which renders the application hands-on
    • instructions on the in app browser

Map/Location finder screen:

  • When there is no data; empty screen would be shown giving an option (+ button on the navigation bar) to bookmark the location.
  • Once the bookmark is added; the you can access 5 days of forecast information upon selecting the bookmark item in the landing (bookmarks list) screen.

Details Screen:

  • You might tap/click on the item(s) listed in the bookmark screen, and so you will be redirected to the details screen.
  • Details screen presents the UI with current location weather information in the 3h time capacity order.
  • You might pull up the view from the bottom of the screen to view the current day and the next consecutive days (5 days forecast) weather report.
  • You might click on any of the weather report summary from the list to view the detailed information.

Settings Screen:

  • Settings screen can be accessed from the home/dashboard screen which can be seen on the top right corner (under the navigation bar)
  • on click of the icon, you will be redirected to the settings screen, where you can change the default preferences.
  • on change of any of the preferences, the weather information will be reflected with these changes.

Help screen:

  • Help screen can be accessed from the floating icons bar under the navigation bar in the landing screen.
  • On click of the icon, you will be redirected to the help screen which provides the application hands-on information.

Reset application data:

  • This option can be accessed from the dashboard floating icons. On click of this icon, all of the bookmarks will be reset.
  • The changes will be persisted immediately on click of these icons.
